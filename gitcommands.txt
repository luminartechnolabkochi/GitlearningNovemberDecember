Basic Git Commands

>git init [initializing git repository in pwd]

>git status [list all untracked files]

>git add -A [adding all untracked files to staging area]

>git commit -m "commit message"[adding staged files to local repository]

>git remote -v [list all remotes]

>git remote add remote_name remote__repository_path[adding remote repo]

>git push remote_name branch[pushing files from local repo to remote repo]
